# Changelog

All notable changes to CoTend are recorded here. CoTend follows Semantic Versioning for published Plugin candidates.

## [0.1.0-rc.1] - Unreleased

### Added

- Seven-Skill Codex Plugin carrier with one normal initialization and resume entry.
- Project continuity, plan-tree, evidence, review, acceptance, diagnosis-only, and model-handoff governance.
- Deterministic skills-only package with locked source, license notices, and brand assets.
- GitHub Open Beta installation through a personal Codex Marketplace.
- Isolated install, update, remove, recovery, and public-repository lifecycle verification.
- Submission listing draft, three starter prompts, and five positive plus three negative reviewer cases.

### Boundaries

- This is a release candidate, not a stable release.
- Public Plugin Directory submission, approval, and publication have not occurred.
- Codex is the only current platform adapter; other AI development tools are not compatibility claims.
