# CoTend v0.1.0-rc.1

CoTend v0.1.0-rc.1 is the first GitHub Open Beta release candidate. It is a Pre-release and not a stable release.

## What is included

- Seven Codex Skills for initializing or resuming projects, continuous governed development, diagnosis without edits, and independent model handoff.
- Explicit user decision boundaries for product choices, destructive actions, payments, secrets, and public release.
- Project-local continuity records, plan-tree navigation, evidence-backed review, AI UAT, and a separate user acceptance gate.
- A deterministic 41-file skills-only Plugin package with Apache-2.0 first-party licensing and preserved third-party MIT notices.

## Install

Follow the [GitHub Open Beta instructions](../../README.md#install-from-github-open-beta). The release archive is an auditable package artifact; normal beta installation uses the GitHub Marketplace commands in the README.

## Verification

The release asset is built from the locked package contract. The accompanying `.sha256` file records the ZIP digest. Unit tests, repository checks, package verification, and submission-material verification must pass before a draft release can be created.

## Known boundaries

- The Plugin has not been submitted to or published in the Public Plugin Directory.
- Existing-task catalog refresh and complete Desktop lifecycle coverage remain beta work.
- The five positive and three negative submission cases are reviewer contracts; an external directory reviewer has not executed them.
- No CoTend account, hosted backend, API key, database, or MCP server is included.
